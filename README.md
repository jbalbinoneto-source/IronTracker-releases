# IronTracker

**IronTracker** é um app de treino em construção, pensado para registrar treinos de forma simples, rápida e confiável.

A ideia é ser um app **offline first**: seus dados principais ficam disponíveis no próprio dispositivo, sem depender de internet para funcionar no dia a dia. Isso também conversa diretamente com privacidade: o foco é permitir que você registre e acompanhe seus treinos sem transformar tudo em nuvem, conta, login, rastreamento e dependência externa.

## O que o app busca resolver

O IronTracker nasceu para quem quer acompanhar treino com menos distração e mais controle:

- registrar sessões de treino;
- acompanhar exercícios, séries, cargas e evolução;
- manter histórico de execução;
- funcionar bem no celular, de forma direta;
- evitar telas poluídas e experiências genéricas de app fitness.

## Em construção

Este projeto ainda está em desenvolvimento. Algumas coisas vão mudar, outras serão ajustadas conforme o uso real.

Opiniões, reviews, críticas e sugestões são muito importantes nesta fase. A intenção é construir o IronTracker de forma horizontal: menos “roadmap perfeito de apresentação” e mais evolução prática com base no que realmente faz sentido para quem usa.

## Privacidade e dados

O IronTracker está sendo pensado com uma lógica simples:

**se o treino é seu, os dados também são seus.**

A prioridade é manter uma experiência local, privada e funcional, sem exigir internet para tudo e sem transformar o app em mais uma plataforma pesada e invasiva.
